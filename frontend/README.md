# Frontend Dashboard (Next.js)

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Current modules

- QAOps overview screen
- Metrics cards (pass rate, coverage, defect density, build stability, release risk)
- Delivery flow visualization (DEV → QA → UAT → PROD)

## Next modules

- Real-time API integration with backend
- Pipeline stage timeline
- Preview environment board per pull request
- Observability links (Grafana, Loki, Jaeger)
